# 💰 Modern Finance Dashboard with Insights & Role-Based UI

A modern and responsive **Finance Dashboard** built using React.js and Tailwind CSS to help users track, analyze, and understand their financial activities.

This project demonstrates clean UI design, component-based architecture, and effective state management while simulating real-world financial tracking.

## 📌 Features

### 📊 Dashboard Overview
- Real-time financial summary (Income, Expenses, Balance)
- Recent transactions display
- Category-wise spending visualization
- Fully responsive design

---

### 💳 Transaction Management
- Add new transactions (Admin only)
- View transaction history
- Category-based organization
- Date sorting and filtering

---

### 🔐 Role-Based UI (RBAC Simulation)
- **Viewer** → Can only view dashboard and transactions  
- **Admin** → Full access, can add transactions  

👉 Role switching implemented using frontend state

---

### 📈 Insights Section
- Total expenses overview
- Highest spending category detection
- Basic financial insights from transaction data

---

### 📉 Data Visualization
- Category-wise expense charts
- Income vs Expense comparison
- Dynamic updates based on data

---

## 🧠 Key Concepts Implemented
- React Hooks (`useState`, `useEffect`)
- Component-based architecture
- Role-based conditional rendering
- LocalStorage for persistence
- Dynamic mock data
- Reusable UI components

---

## 🛠️ Tech Stack
- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **Charts:** Chart.js  
- **Icons:** React Icons
